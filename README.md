# Petite description du projet :  
Bonjour, bienvenue sur le super projet de l'équipe Java script. Ce projet a été réalisé pour le client Jean Révise.  
Le projet consiste en un projet pour pouvoir faire ses propres flashcards et pouvoir, les réviser et s'entraîner dessus.

***
# Comment lancer le projet :  
Il y a 2 fichiers pour lancer le projet :  
* Main.py -> lance le programme en version interface console.
* Maingui.py -> lance le programme en version interface graphique utilisateurs.

Au lancement du projet, 4 choix sont proposés :  
* Quiz -> pour faire un quiz sur les cartes créé au préalable.
* flashcard -> pour réviser ses cartes créées au préalable
* Gestion des chapitres -> pour pouvoir gérer des chapitres et y mettre des cartes.
* quitter -> pour fermer le projet
* 
***
# Module à installer avant d'utiliser le projet :  
Le projet utilise des les modules externes qui nécessitent d'être installés pour le bon fonctionnement du projet.
* `py -m pip install pillow` -> nous utilisons ce module pour pouvoir mettre des images dans les cartes en mode Interface Graphique Utilisateur.
